.Errors
--
.422 unprocessable entity

Incorrect data in request.
User's email should contains of:

 * symbol '@'
 * at least 1 symbol before '@';
 * at least 1 symbol after '@';
 * symbol '.' after previous point;
 * after '.' should be between 2 and 4 symbols;

User's  username length should be between 3 and 50;

User's  password length should be between 6 and 255;
